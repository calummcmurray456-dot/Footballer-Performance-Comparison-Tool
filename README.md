# Football Performance Analysis

A football analytics dashboard covering talent scouting, player comparison, and injury risk prediction across 30+ leagues worldwide.

## Features

- **Talent Scouting** – Identifies high-potential players using a Random Forest model trained on historical performance data.
- **Player Comparison** – Compares any two players using cosine similarity across position-weighted stats, with radar chart visualisation.
- **Injury Risk Prediction** – Predicts injury likelihood using workload and historical injury data.

## Tech Stack

- **Python** with **Streamlit** for the interactive dashboard
- **pandas** / **NumPy** for data processing
- **scikit-learn** for machine learning models (Random Forest, cosine similarity)
- **Plotly** for data visualisation (radar charts, scatter plots)
- **BeautifulSoup4** / **requests** for web scraping (FBref, Transfermarkt)

## Data Sources

- **FBref** – player and team performance statistics
- **Transfermarkt** – injury history and records
